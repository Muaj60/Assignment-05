1️⃣ What is the difference between var, let, and const?

Ans:var is the old way to declare variables. It can be redeclared and updated.
let is a newer way. It can be updated but cannot be redeclared in the same scope.
const is used for variables that should not change. It cannot be updated or redeclared.

2️⃣ What is the spread operator (...)?

Ans:The spread operator  is used to copy or expand elements of an array or object.
It helps combine arrays or copy values easily.

3️⃣ What is the difference between map(), filter(), and forEach()?

Ans:map() → creates a new array by changing each element.
filter() → creates a new array with elements that match a condition.
forEach() → runs a function for each element but does not return a new array.

4️⃣ What is an arrow function?

Ans:An arrow function is a short way to write a function in JavaScript.
It uses => instead of the function keyword.

5️⃣ What are template literals?

Ans:Template literals are used to create strings easily and add variables inside strings.
